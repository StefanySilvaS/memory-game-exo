# 🎮 Memory Game EXO

Um jogo da memória divertido inspirado no grupo EXO, desenvolvido com HTML, CSS e JavaScript.

🔗 **Jogue agora:** https://stefanysilvas.github.io/memory-game-exo/

---

## 🚀 Sobre o projeto

Este projeto é um jogo da memória clássico onde o objetivo é encontrar todos os pares de cartas iguais.

O jogador precisa usar a memória para virar as cartas e combinar os pares no menor número de tentativas possível.

---

## 🧠 Como jogar

1. Clique em uma carta para revelar
2. Clique em outra carta para tentar formar um par
3. Se forem iguais, o par permanece aberto
4. Se forem diferentes, as cartas viram novamente
5. Continue até encontrar todos os pares 🎉
